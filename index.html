Getting Started with ReviewAI

keyStep 1 — Get Your Free Groq API Key

  1. Visit console.groq.com
  2. Sign up (no credit card needed)
  3. Go to API Keys → Create API Key
  4. Copy your key

editStep 2 — Add Key to This File

Open index.html in any text editor and find this line near the top of the <script> section:

const GROQ_API_KEY = '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';

Replace YOUR_GROQ_API_KEY_HERE with your actual key. Save the file.

rocket_launchStep 3 — Deploy for Free

  1. Go to netlify.com and sign up free
  2. In the dashboard, drag & drop your index.html file
  3. Netlify gives you a live URL instantly!

Alternatively: Vercel or GitHub Pages also work.
